# butter

A Simple beautiful Habit Tracker App

android build apk

```
flutter build apk --split-per-abi
```

build icons for app

```
flutter pub run flutter_launcher_icons
```

generate model

```
flutter pub run build_runner build
```

for splash page

```
flutter pub run flutter_native_splash:create
```
